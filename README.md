# Azizi Tarany Starter

Contains the first 10 Pashto audio tracks, instant search, play/pause, and a progress slider.

## Run
1. Install Flutter and Android tooling.
2. In this folder run: `flutter create .`
3. Run: `flutter pub get`
4. Connect an Android phone with USB debugging enabled or start an emulator.
5. Run: `flutter run`

## Build APK
`flutter build apk --release`

APK output is normally under `build/app/outputs/flutter-apk/`.
